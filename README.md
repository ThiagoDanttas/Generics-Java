# Generics - Java

---
Generics permitem que classes, interfaces e métodos possam ser
parametrizados por tipo. Seus benefícios são:

• Reuso

• Type safety

• Performance

• Uso comum: coleções

~~~~java
public class Program {
    List<String> list = new ArrayList<>();
    list.add("Maria");
    String name = list.get(0);
}
~~~~
